# 📖 Quran App UI
A clean and elegant Flutter application that brings together the beauty of the Quran in a single place — Names of Allah, Names of the Prophet (S.A.W), Surahs, Juz', Sajdas, and Manzils — all fetched live from the AlQuran Cloud API.

## ✨ Features
This app dynamically fetches and displays the following data from the AlQuran Cloud API:

### 🕌 Names of Allah — All 99 beautiful names (Asma-ul-Husna) with meanings
### 🌙 Names of Prophet Muhammad (S.A.W) — Blessed names with their meanings
### 📜 Surah Names & Details — Full list of 114 Surahs with Arabic names, translations, revelation type, and ayah count
### 📚 Juz' Names & Details — All 30 Juz' (Para) with relevant info
### 🛐 Sajda Details — Locations and information of all Sajda verses in the Quran
### 📿 Manzil Names & Details — The 7 Manzils for structured daily recitation

## 🛠️ Built With
#### Flutter — Cross-platform UI toolkit
#### Dart — Programming language
#### AlQuran Cloud API — Quran data source
#### HTTP Package — For API requests
#### Provider State Management
